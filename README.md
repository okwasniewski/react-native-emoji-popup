# react-native-emoji-popup

Emoji Popup for React Native

## Installation

```sh
npm install react-native-emoji-popup
```

## Usage


```js
import { EmojiPopupView } from "react-native-emoji-popup";

// ...

<EmojiPopupView color="tomato" />
```
